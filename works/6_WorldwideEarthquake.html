Worldwide Earthquake

D3/P5.js/Physical interaction Practice

Description

A data visualization project shows earthquakes around the world in the last hour.
This project includes a website and a physical interactive installation.
This website is based on d3.js and p5.js.

(Recommended to be viewed on laptop for best views)

This interactive installation module is based on Microbit.

Website

Based Data

Source data: earthquake.usgs.gov.

inspiration

It is inspired by the collapse and amplitude of earthquakes that occur.
This website is based on d3.js and p5.js.

(Recommended to be viewed on laptop for best views)

Interactive Installation

When the latest earthquakes occur in the world,
this interactive installation will vibrate
with an intensity determined by the magnitude of the earthquake.
